## Films online 
A small application Where you can find films you need.
You can also can see all description and more about film 


## How to start
* `git clone https://github.com/ildevelop/remyfilms.git`  download repository
* `cd remyfilms` go inside the main folder
* `npm i`  download all libraries in dependencies
* `npm start`  start the project

##Feature
* live updating query request
* search films by  `[DOMAIN]?q=[MOVIE_NAME]`
* search film by id `[DOMAIN]/(imdbID)`
* pagination
* Pure JS + CSS without external theming library
